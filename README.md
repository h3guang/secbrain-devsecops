# SecBrain · AI 驱动的 DevSecOps 智能安全流水线

> 代码提交即扫描 · AI 自动分级 · 让漏洞修复从 7 天缩短到 4 小时

---

## 📌 项目简介

**SecBrain** 是一个嵌入 GitHub Actions 的智能安全扫描流水线。开发人员提交代码后，自动触发安全扫描，并通过 AI 决策引擎对漏洞进行 **P0/P1/P2 三级分类**，让开发人员秒懂优先级、秒拿修复方案。

**一句话说清：** 开发人员正常 `git push`，SecBrain 在后台自动扫描、AI 分级、给出修复建议，让安全不拖慢开发节奏。

---

## 🎯 解决什么问题

| 传统痛点 | SecBrain 的解法 |
|----------|-----------------|
| 安全扫描在部署后才跑，发现问题要回退代码 | **提交即扫描**，代码合并前就发现问题 |
| 漏洞报告是 CVE 编号 + 英文技术描述，开发看不懂 | **AI 翻译成人话**，P0/P1/P2 中文分级 |
| 开发不知道怎么修，自己研究半天 | **AI 直接给修复建议**，复制粘贴就能修 |
| 所有漏洞同等对待，一刀切卡死发布 | **分级阻断**：P0 阻断，P1 建议修复，P2 允许上线 |
